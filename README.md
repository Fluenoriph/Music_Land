Проект Music Land. 

  Это в большей мере база данных музыкальных файлов, реализованная на PostgreSQL.
  Состав информации, относящийся к одному звуковому файлу:
  1. Имя файла
  2. Размер файла (мб)
  3. Расположение (директория)
  4. MD5 хэш-сумма файла
  5. Исполнитель
  6. Название трека
  7. Альбом
  8. Жанр
  9. Год
  10. Кодек
  11. Битрейт (кбит/сек)
  12. Частота дискретизации (кГц)
  13. Время трека (мин)
  14. Разрядность (бит)
  15. Формат звука (mp3, flac и т.п.)

  Реализован механизм записи в базу данных всех найденных звуковых файлов в директории. А также единичного файла. С проверкой по хэшу на существование в базе.
